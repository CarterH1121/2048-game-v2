# 2048 后台管理系统界面设计规范

> 绘境设计 | Hermes团队 | 版本 1.0

---

## 1. 设计理念

### 1.1 核心理念

**「智感后台，清晰掌控」**

以科技感为基调，融合2048游戏的品牌基因，打造专业、高效、美观的后台管理体验。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| **清晰优先** | 信息层次分明，关键数据一目了然 |
| **高效交互** | 减少操作步骤，提升工作效率 |
| **视觉统一** | 组件风格一致，品牌识别度高 |
| **响应适配** | 支持多端访问，布局自适应 |

### 1.3 设计关键词

```
科技感 | 专业 | 现代 | 高效 | 数据驱动
```

## 4. 组件规范

### 4.1 按钮

#### 按钮类型

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主要按钮 | 品牌色填充 | 主要操作（保存、确认） |
| 次要按钮 | 边框样式 | 次要操作（取消、返回） |
| 文字按钮 | 无边框 | 链接跳转、表格操作 |
| 图标按钮 | 正方形 | 工具栏、更多操作 |

#### 按钮尺寸

```
大按钮: height: 44px, padding: 0 24px, radius: 8px
中按钮: height: 36px, padding: 0 16px, radius: 6px
小按钮: height: 28px, padding: 0 12px, radius: 4px
```

## 8. 动效规范

### 8.1 过渡时间

| 类型 | 时长 | 缓动函数 |
|------|------|----------|
| 快速 | 150ms | ease-out |
| 标准 | 200ms | ease-in-out |
| 慢速 | 300ms | ease-in-out |
| 弹性 | 400ms | cubic-bezier(0.34, 1.56, 0.64, 1) |

### 8.2 常用动效

#### 悬停效果

```css
/* 按钮悬停 */
transition: all 150ms ease-out;

/* 卡片悬停 */
.card:hover {
  transform: translateY(-2px);
  box-shadow: 0 10px 25px -5px rgba(0, 0, 0, 0.3);
}

/* 表格行悬停 */
transition: background-color 150ms ease-out;
```

#### 弹窗动画

```css
/* 遮罩层 */
.modal-overlay {
  animation: fadeIn 200ms ease-out;
}

/* 弹窗主体 */
.modal {
  animation: slideUp 300ms cubic-bezier(0.16, 1, 0.3, 1);
}

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

#### 数字变化动画

```css
/* 数字增长动画 */
.number-animate {
  transition: all 500ms ease-out;
}

/* 加载动画 */
.loading-spinner {
  animation: spin 1s linear infinite;
}

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}
```

### 8.3 动效原则

| 原则 | 说明 |
|------|------|
| **目的性** | 动效要服务于功能，不纯粹装饰 |
| **克制性** | 避免过度动效，影响性能和体验 |
| **一致性** | 相同交互使用相同动效 |
| **可关闭** | 提供关闭动效的选项 |
